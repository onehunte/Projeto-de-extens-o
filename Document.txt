## Visão Geral
O Proteplan viwer é uma aplicação desktop desenvolvida em Python usando o framework Kivy, projetada para exibir e gerenciar uma biblioteca de ebooks. A aplicação se conecta a um servidor Flask para recuperar informações sobre os ebooks disponíveis e permite que os usuários abram os arquivos PDF diretamente do aplicativo.
